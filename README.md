# Hotel Management System (Python)

This is a simple Python program that demonstrates basic hotel management operations using classes and object-oriented programming.

## Features

- Store hotel information
  - Hotel name
  - Available rooms
  - Location
  - Rating
  - Room price
- Store user booking details
- Display all hotel records
- Sort hotels by:
  - Name
  - Rating
  - Available rooms
- Search hotels by city
- Display user booking information

## Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
